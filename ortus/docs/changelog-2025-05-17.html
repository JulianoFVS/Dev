Release Notes ORTUS — 17/05/2025

ORTUS

Sistema de Gestao Odontologica Inteligente

Release Notes
17 de Maio de 2025

Esta atualização traz 11 melhorias significativas ao sistema ORTUS, incluindo um modulo completo de Harmonizacao Orofacial, relatorios gerenciais, busca global inteligente e migracao de dados para a nuvem. Todas as alteracoes foram validadas com verificacao de tipos (TypeScript).

Novo Modulo

1. Harmonizacao Orofacial (HOF)

Onde acessar: Ficha do Paciente → aba "Harmonizacao (HOF)"

  • Mapa facial interativo com marcacao de pontos de aplicacao
  • 5 tipos de marcacao: Toxina Botulinica, Acido Hialuronico, Bioestimulador, PMMA, Fios PDO
  • Upload de fotos antes/depois por angulo (Frontal, Perfil D, Perfil E, Sorrindo)
  • Alertas inteligentes de vencimento de toxina (90-180 dias)
  • Protocolos pre-definidos (Harmonizacao Completa, Bigode Chines, Olheiras, Rinomodelacao, etc.)
  • Termo de Consentimento HOF com aceite e impressao
  • Impressao do Mapa HOF em documento A4 formatado
Melhoria

2. Armazenamento em Nuvem (Storage)

Onde acessar: Aba HOF (fotos) e aba "Documentos"

  • Fotos e documentos agora sao enviados para Supabase Storage (nao mais base64)
  • Compressao automatica de imagens (max 1200px, JPEG 80%)
  • Exclusao remove arquivo fisico do bucket
  • Limite de upload aumentado de 3MB para 10MB
Correcao

3. Inbox Reativo

Onde acessar: Pagina /inbox

Corrigido: marcar notificacao como lida agora atualiza a lista instantaneamente, sem recarregar a pagina inteira.

Novo Recurso

4. Aniversariantes no Dashboard

Onde acessar: Dashboard — secao rosa/ambar no topo

  • Card com aniversariantes da semana (proximos 7 dias)
  • Indicador pulsante para aniversario de hoje
  • Idade calculada automaticamente
  • Botao WhatsApp com mensagem de felicitacao pre-formatada
Novo Recurso

5. Confirmacao via WhatsApp na Agenda

Onde acessar: Agenda → clicar em qualquer agendamento

  • Botao verde "Confirmar via WhatsApp" no modal de agendamento
  • Aparece automaticamente quando paciente tem telefone cadastrado
  • Mensagem com nome, data, horario e procedimento
Novo Recurso

6. Omnibar — Busca Global (Ctrl+K)

Onde acessar: Qualquer pagina → Ctrl+K ou botao "Buscar..." na sidebar

  • Busca pacientes por nome, CPF ou telefone
  • Navegacao rapida para qualquer pagina do sistema
  • Teclado: setas para navegar, Enter para selecionar, Esc para fechar
  • Clicar em paciente abre o Slide-Over de detalhes
Nova Pagina

7. Pagina de Relatorios

Onde acessar: Sidebar → "Relatorios" (apenas administradores)

  • KPIs: Receita Total, Despesas, Lucro Liquido, Pacientes Atendidos
  • Taxa de Comparecimento com grafico circular
  • Valores em aberto (fiados) com alerta visual
  • Ticket Medio por consulta
  • Grafico de Faturamento Mensal (barras)
  • Ranking de Procedimentos mais realizados
  • Filtros: Este mes, 3 meses, 6 meses, Este ano
  • Botao Imprimir relatorio
Novo Recurso

8. Evolucao / Prontuario Clinico

Onde acessar: Ficha do Paciente → aba "Evolucao"

  • Registro narrativo de cada atendimento (obrigatorio CRO)
  • Campos: data, profissional, descricao livre
  • Timeline cronologica visual com todas as evolucoes
  • Botao "Imprimir Prontuario" — documento A4 com campo de assinatura
  • Excluir registros individualmente
Melhoria

9. Refatoracao Interna (page.tsx paciente)

Melhoria interna: a aba Evolucao foi extraida como componente independente (TabEvolucao.tsx), reduzindo ~90 linhas do arquivo principal e estabelecendo padrao para futuras modularizacoes.

Infraestrutura

10. Migracao localStorage → Supabase

Impacto: Configuracoes e Financeiro agora sincronizam entre dispositivos

  • Categorias financeiras persistidas no banco por clinica
  • Preferencias da clinica (horarios, dias, tema) no banco
  • Modelos de documentos (contratos, termos) no banco
  • Meta de lancamentos (cancelamentos) no banco
  • Fallback automatico: dados antigos do localStorage migram sozinhos
  • Nova tabela: configuracoes_clinica com RLS
Novo Recurso

11. Busca Avancada de Pacientes

Onde acessar: Pagina /pacientes → botao "Filtros"

  • Busca agora inclui CPF alem de nome e telefone
  • Filtro por Status (Ativo / Agendado / Novo)
  • Filtro "Com debito" — apenas pacientes devedores
  • Filtro "Sem consulta ha X dias" (30, 60, 90, 180, 365)
  • Filtro por procedimento pendente (busca textual)
  • Indicador visual quando filtros estao ativos
  • Botao "Limpar filtros"
